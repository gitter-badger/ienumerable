# Api Reference

As described before IEnumerable exports only one class to do its work. You can import it like this:

#### ES6

```js
import Enumerable from 'ienumerable'
```

#### ES5 (CommonJS)

```js
var Enumerable = require('ienumerable').default
```

### Api

Api are divided 4 categories:

- [Creational](Creational/README.md)
- [Converters](Converters/README.md)
- [Sql Like](SqlLike/README.md)
